<p id="description">A fully responsive and modern website for Azkaban e-Esports built using React.js and Vite. It showcases team rosters esports achievements and organization details with a dark-themed UI. Designed for speed and performance the site offers smooth navigation reusable components and a professional layout optimized for both desktop and mobile users.</p>

<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Fully responsive design (desktop + mobile)
*   Built with React.js and Vite
*   Dynamic routing with React Router
*   Collapsible and interactive sidebar
*   Separate roster sections for different esports teams
*   Hall of Fame and Achievements showcase
*   Clean and professional layout
*   Smooth navigation and fast performance
*   Modular and reusable components

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/gravityyyy99/AZKABAN-e-Sports.git
```

<p>2. Navigate into the project folder</p>

```
cd AZKABAN-e-Sports
```

<p>3. Install dependencies</p>

```
npm install
```

<p>4. Start the development server</p>

```
npm run dev
```
